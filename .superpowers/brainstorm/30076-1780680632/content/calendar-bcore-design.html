B-core 设计草案：日历排期底座

目标：一期建立真正可排期的日历，而不是只把任务按日期画成格子。

数据流
Task 增加 schedule 字段：plannedDate / startAt / endAt / allDay
Tasks API 增加日期范围查询与排期更新
Calendar Controller 负责视图切换、筛选和显示设置
Calendar Panel 渲染月 / 周 / 列表
Focus Sessions 作为专注记录 overlay
用户界面
月视图 · 周视图 · 列表视图 · 今天
左：无时间任务 / 分类筛选
右：日历主体 / 时间轴
显示设置：范围 / 已完成 / 分类颜色 / 专注记录

一期交互边界

1

月视图

按日期格显示任务；点击日期创建任务；拖任务到日期改变 plannedDate。

2

周视图

全天栏 + 竖向时间轴；有 startAt/endAt 的任务进入时间轴；无时间任务在全天栏或侧栏。

3

列表视图

按天分组显示任务，支持显示时间轴摘要，适合轻量浏览。

4

基础显示设置

显示范围、显示已完成、分类颜色、专注记录。暂不做农历、节假日、辅助时区、批量推迟。
